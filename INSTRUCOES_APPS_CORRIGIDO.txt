Como aplicar:
1. Com o servidor parado, renomeie sua pasta apps atual para apps_antes_correcao.
2. Extraia este ZIP na raiz do projeto, de forma que exista novamente a pasta apps.
3. Rode:
   python manage.py showmigrations --plan
   python manage.py migrate
   python manage.py runserver

Observação:
As correções cobrem os conflitos identificáveis pelos arquivos do projeto e pelos erros já apresentados.
Se o banco tiver outras tabelas ou colunas criadas sem registro no django_migrations, o migrate poderá revelar novo ponto de divergência.
